# KaleidoBrands Executive Dashboard

The Executive Dashboard provides a central view of KaleidoBrands business operations.

## Included Metrics

- Total quotes
- Total orders
- Total revenue
- Open CRM pipeline value
- Recent quotes
- Recent orders
- Recent CRM leads
- Recent supplier sync logs

## Charts

- Orders by status
- CRM pipeline value
- Supplier sync activity
