# KaleidoBrands CRM

The CRM module helps KaleidoBrands manage customer relationships, leads, follow-ups, and sales opportunities.

## Features

- Customer leads
- Lead status pipeline
- CRM activity tracking
- Follow-up reminders
- Sales pipeline dashboard
- CRM analytics
- Auto-created leads from Quote Builder

## Lead Statuses

- New Lead
- Contacted
- Qualified
- Proposal Sent
- Negotiation
- Won
- Lost

## Management Commands

Send follow-up reminders:

```bash
python manage.py send_crm_followups
